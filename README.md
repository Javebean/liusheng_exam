#Usage
relax , it's very easy!

```javascript
 var rotate = new Rotate({
    	canvasHeight:500,
    	canvasWidth:300,
    	canvasBgcolor: '#fff',
    	canvasBorderWidth: '1px',
    	canvasBorderColor: 'red',
    	imageSrc:'pic/hah.png'
    });
    
    $("#clockwise").click(function(){ 
        rotate.clockwise();
    });

    $("#counterclockwise").click(function(){ 
        rotate.counterclockwise();
    });
```
